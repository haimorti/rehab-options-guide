# content-and-spec

תיקייה זו מחליפה את `pdf-source` ומכילה את חומר המקור למסמך "מסלולי שיקום מקצועי" החדש.

## קבצים

- `01-structure-and-navigation.txt` — מפרט מבנה, מטרת המסמך, היררכיית העמודים ודרישות הניווט הפנימי ב-PDF.
- `02_content.txt.txt` — תוכן הטקסט המלא לפי עמודים. לפי ההנחיה בקובץ המקור, אין לשנות את ניסוח התוכן.

## מה למדתי מהתיקייה

- המוצר החדש הוא PDF לחיץ שמתנהג כמו אתר קטן, אך נשאר קובץ יחיד ונייד.
- המסמך מיועד למשתקמים שאושרה להם זכאות לשיקום מקצועי, כדי שיגיעו לפגישה עם עובד/ת השיקום מוכנים יותר.
- המבנה כולל 11 עמודים בשלוש רמות: עמוד בית, עמודי מסלול, ועמודי תוכן סופיים.
- חובה לתכנן ניווט פנימי: חזרה לבית, חזרה לעמוד ההורה, וסימון מיקום.
- התוכן מחולק לארבעה מסלולים ראשיים: אבחון והתאמת תוכנית, טרום שיקום, לימודים והכשרה, והשמה בעבודה.
